# LYMINA Website

Website of our band LYMINA

**Live Site:** [lymina.de](https://lymina.de)

## Local Development

1.  Start a local server using Python:
    ```bash
    python3 -m http.server 8000
    ```
2.  Open the site in your Chrome browser:
    ```bash
    google-chrome http://localhost:8000
    ```
